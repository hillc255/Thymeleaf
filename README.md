# Thymeleaf
Thymeleaf, a Spring Boot template engine  - Udacity Java Nanodegree.

## Tools used :

    Spring Boot2.1.8.RELEASE
    Spring Web dependency
    Thymeleaf
    Maven
    Embedded Tomcat
    
 ## Run on command line  
 
    mvn spring-boot:run 
    
 ## In browser  
   
   http://localhost:8080/demo

   Hello Thymeleaf
